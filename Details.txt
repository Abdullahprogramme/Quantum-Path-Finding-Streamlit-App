QUANTUM NAVIGATOR — Full Project Plan
══════════════════════════════════════════════════════════════

THEME: "Three Minds, One Maze"
  - A Blind Mind   → BFS  (explores everything, no knowledge)
  - A Smart Mind   → A*   (uses heuristic knowledge)
  - A Quantum Mind → Grover's (uses superposition, all paths at once)

GRID DESIGN:
  - N×N grid (N slider: 5 to 20)
  - Each cell = a node
  - Obstacles = black cells (user places them or auto-random)
  - Start = green cell, End = red cell
  - Paths highlighted in 3 different colors per algorithm

SIDEBAR CONTROLS:
  - Grid size slider (N: 5–20)
  - Obstacle density slider (0–40%)
  - Random seed input
  - [Apply Settings] button
  - [Run All Algorithms] button
  - Toggle: Show/Hide quantum circuit diagram

MAIN PANEL (right):
  - Top: N×N interactive grid (Streamlit)
  - Below grid: 3-column benchmark table
  - Below benchmarks: Matplotlib figures
      → Quantum circuit diagram
      → Qubit probability histogram
      → Convergence chart (amplitude per Grover iteration)
      → Path length comparison bar chart

QUANTUM LOGIC (Grover's):
  - Grid nodes encoded as binary strings → qubits
  - Oracle marks the TARGET node's binary encoding
  - Diffusion operator amplifies target amplitude
  - Optimal iterations = floor(π/4 × √N²)
  - Simulate with Qiskit Aer (statevector_simulator)
  - Extract highest-probability state → decode to grid path
  - Path reconstruction: trace back using BFS from decoded node
    (Quantum finds the TARGET; path is reconstructed classically)

CSP INTEGRATION:
  - Obstacles are encoded as forbidden states in the Oracle
  - Oracle flips phase only for valid, non-obstacle target states
  - This is the Constraint Satisfaction layer

PROBABILISTIC REASONING:
  - Full statevector extracted after simulation
  - Histogram shows probability of each basis state
  - Shows quantum uncertainty visually

PERFORMANCE MEASURES:
  - Time (ms) for each algorithm
  - Path length (number of cells)
  - Nodes explored / states evaluated
  - Memory usage estimate
